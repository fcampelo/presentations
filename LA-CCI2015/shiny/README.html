Towards Statistical Convergence Criteria for Mutation-based EAs

This is a small application intended at illustrating some concepts related to the paper:

F. Campelo, “Towards Statistical Convergence Criteria for Mutation-based Evolutionary Algorithms”, 2nd Latin American Congress on Evolutionary Computation - LA-CCI, Curitiba, Brazil, 2015.

In this example a simple \((1 + \lambda)\)-ES is applied to an n-dimensional spherical function with \(x_i \in \left[-0.5, 0.5\right],\ i = 1, \ldots, n\).

This app was created using Shiny, which is a web application framework for R developed by Winston Chang and the fine folks from RStudio. The source code for the app will be available from my Github account.

This README page was also written in RStudio, using R Markdown.

I hope you enjoy this little demo. Comments and suggestions are all very welcome.
